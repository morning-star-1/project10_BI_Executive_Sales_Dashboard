# Executive Sales Dashboard (Power BI)

A Power BI executive sales dashboard for business stakeholders, built from a small sample sales dataset.

## Screenshot
![Screenshot](docs/screenshot.png)

Replace `docs/screenshot.png` with a real screenshot or GIF.

## Quickstart
### Prerequisites
- Power BI Desktop (Windows)

### Open the dashboard
Open `powerbi/Executive_Sales_Dashboard.pbix` in Power BI Desktop.

## Data
- Sample data: `data/raw/sales.csv`

## Exports
- `powerbi/Executive_Sales_Dashboard.pdf` is a static export of the dashboard.

## Tests

python -m unittest discover -s tests
